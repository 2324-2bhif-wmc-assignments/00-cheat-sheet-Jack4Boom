Lists and Tables

Lists

Different Start tags

  • <ol> for ordered list, list items will be marked with numbers or letters
  • <ul> for unordered list, list items will be marked with a dot or square
  • <li> for list item

Css for lists

  • list style type: square/disc/circle; Sets the marker of a list element
  • list-style-type: upper-roman; Sets marker to uppercase roman numbers
  • list-style-image: url("my-bullet.png"); Sets the marker of a list element with an image
  • list-style-position: inside; Sets markers position to be inside the list item's text flow
  • list-style-position: outside; Sets markers position to be outside the list item's text flow

Tables

Start tags for Tables

  • <table> Start of a table
  • <tr> a table row
  • <theader> Table header cell
  • <tbody> Table body
  • <td> Table cell
  • <tfooter> Table footer
  • <td> Table Data

Css for Tables

  • border: 1px solid black; Creates a black dotted border
