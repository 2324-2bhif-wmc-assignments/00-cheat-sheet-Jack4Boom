World Wide Web

What are the most essential technologies that made the World Wide Web such a success?

    URL (Uniform Resource Locator) : serves as a system for resources on the web.

    HTTP (HyperText Transfer Protocol) : specifies communication of browser and server.

    HTML (Hyper Text Markup Language) : defines the structure, organisation and content of a webpage.

How are resources (html pages, images, etc) addressed in the web?

    They have an URL for themselves

What are the general components of a url?

    The scheme : identifies the protocol to be used to access the recourse on the internet (http or https)

    A host : identifies the host that holds the resource (www.goggle.com, google would be the host)

    A path : identifies the specific resource in the host that the web client wants to access (/documents/main/index.html)
