World Wide Web

What are the most essential technologies that made the World Wide Web such a success?

    URL (Uniform Resource Locator) : serves as a system for resources on the web.

    HTTP (HyperText Transfer Protocol) : specifies communication of browser and server.

    HTML (Hyper Text Markup Language) : defines the structure, organisation and content of a webpage.

How are resources (html pages, images, etc) addressed in the web?

    They have an URL for themselves

What are the general components of an url?

    The scheme : identifies the protocol to be used to access the recourse on the internet (http or https)

    A host : identifies the host that holds the resource (www.goggle.com, google would be the host)

    A path : identifies the specific resource in the host that the web client wants to access (/documents/main/index.html)

 

 

Web Server

What is a web server?

    A web server is computer software and underlying hardware that accepts request via http or the secure variant https. Usually a web browser initiates communication through a request for a specific resource using http/s. The server either responds with the resource of the request or with an error message

What does a web server?

    Web server process and manage http/s requests and responds. A web server cam also store and protect critical website data from unauthorized users

How can content be stored on a web server?

    They use some kind of database (server-side storage), then run server-side code to retrieve needed data

How is content in webpages described?

    Usually as text, but it can also be described as links, audios, animations, etc

 

 

Web Client

What is a web client?

    A web client or Web browser is an application for accessing websites

What does a web client?

    The Purpose of a web client is to fetch content from the world wide web or from local storage and display it on the user's device.

Is Web Client and Web Browser is the same?

    Yes it is the same thing
