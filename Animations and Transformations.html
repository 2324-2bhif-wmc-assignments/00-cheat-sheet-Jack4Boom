Animations and Transformations

Animations

Properties of animations

  • @keyframes: keyframes are needed as they are the name of the animation
  • animation-name: keyframe name needs to be inserted so right animation is chosen
  • animation-duration: Time span how long animation play from start to finish
  • animation-delay: Delay when animations starts
  • animation-iteration-count: How often animation should be played, can also be infinite
  • animation-direction: In which direction animation plays. If alternate is used it will play animation and then play animation in reverse

Transformations

Transformation properties

  • Transform: rotate("number"deg): Rotates number degrees
  • Transform: scale(number): Resizes element based on number
  • Transform: translate(x, y, z): First number is x-axis, second number is y-axis and third number is z-axis; It defines position at the x/y/z axis
