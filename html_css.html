Html and Css basics

Basics

Components of html

  • First of all a start tag like <div>

  • Then there can be attributes like href=""

  • Then theres content after the start tag like <div> hello </div>

  • And then there is the end tag </div> to close the element

Void elements

  • Void elements only have start, but no end tags

  • Void elements can not have content inside of it

  • void elements have attributes such as <br>. It accepts the clear attribute that indicates where to start the next line

  • void elements can not be nested (meaning they cant contain other elements)

Attributes

  • Attributes provide additional content information about html elements

    • href for url's

    • src for the path of images

    • alt is alternative text if img can not be loaded

    • style is used to add style to an element, such as color, font, size and more

!Doctype html

  • Is a Document declaration for html

Top level element

  • The Html document starts and ends with the <Html> tag

Body

  • The visible part of the website is between the <body> tag

Line break

  • The tag for line break is <br>

Section with meta information

  • <meta charset="UTF=8">

Headings

  • These are headings. h1 is the biggest and h6 is the smallest

Paragraphs

  • paragraphs are defined with the <p> tag

Links

  • Links are defined like this: <a href="orf.at">click here</a>

Page label

  • <title>Title for page</title>

Components of Css

  • First you have a selector such as <h1> or <body> which decides what element should be styled

  • Then you have properties like font-family and such to change the style

  • At last is the value to put in the properties like arias in font-family or blue in color
