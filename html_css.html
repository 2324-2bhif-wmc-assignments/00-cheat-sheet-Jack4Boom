Html and Css basics

Html

Components of html

  • First of all a start tag like <div>

  • Then there can be attributes like href=""

  • Then theres content after the start tag like <div> hello </div>

  • And then there is the end tag </div> to close the element

Void elements

  • Void elements only have start, but no end tags

  • Void elements can not have content inside of it

  • void elements have attributes such as <br>. It accepts the clear attribute that indicates where to start the next line

  • void elements can not be nested (meaning they cant contain other elements)

Attributes

  • Attributes provide additional content information about html elements

    • href for url's

    • src for the path of images

    • alt is alternative text if img can not be loaded

    • style is used to add style to an element, such as color, font, size and more

!Doctype html

  • Is a Document declaration for html

Top level element

  • The Html document starts and ends with the <Html> tag

Body

  • The visible part of the website is between the <body> tag

Line break

  • The tag for line break is <br>

Section with meta information

  • <meta charset="UTF=8">

Headings

  • These are headings. h1 is the biggest and h6 is the smallest

Paragraphs

  • paragraphs are defined with the <p> tag

Links

  • Links are defined like this: <a href="orf.at">click here</a>

Page label

  • <title>Title for page</title>

Css

Components of Css

  • First you have a selector such as <h1> or <body> which decides what element should be styled

  • Then you have properties like font-family and such to change the style

  • At last is the value to put in the properties like arias in font-family or blue in color

Color by color name or hexadecimal notation

  • You can write color = "black" for expample

  • You can also write color = #000000 (black)

Backround color

  • You can write backround-color = "white"

  • You can also write backround-color = #FFFFFF

Pseudo classes for the <a> tag

  • a:link {

      color = "black"

    }

  • a:visited{

      color = "red"

    }

  • a:hover{

      color = "blue"

    }

  • a:active{

      color = "yellow"

    }

Backround image

  • <p style ="backround-image": url('htl.jpg')

Text alignment

  • p{

      text-align = left/right/center/justify

    }

Text decoration

  • p{

      text-decoration = overline/underline/line-through

    }

Text transformation

  • p{

      text-transform = uppercase/lowercase/capitalize

    }

Font family

  • p{

      font-family: "Century Gothic"

    }

Font style

  • p{

      font-style: italic

    }

Font size

  • p{

      font-size: em/rem/px/%/small/large

    }

Font weight

  • p{

      font-weight: normal/bold/bolder/lighter/number

    }

Tilde selector

  • The element1~element2 selector matches occurrences of element2 that are preceded by element1

  • Both elements must have the same parent, but element2 does not have to be immediately preceded by element1


    • p ~ ul {

      background: #ff0000;

    }

Direct child

  • Would be the syntax for a direct child meaning it selects all h1's where body is the parent


    body > h1{

    }

Descendant

  • Would be the syntax for a descendant which means select a that are descendents of div, meaning it selects every a inside a div


    div a{

    }

Adjacent sibling

  • The element+element selector is used to select an element that is directly after another specific element


    div + p{

    }
